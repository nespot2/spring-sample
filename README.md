# spring-sample

----

### 개요

- spring 핵심 기능을 사용하여 간단한 쇼핑몰 사이트를 만드는 것을 목표로 합니다.

### stack

#### front-end

- fxjs
- fxdom
- thymeleaf


#### back-end

- spring boot
- spring jpa wiht query dsl
- spring mvc
- spring security


#### 인프라

- ec2 with vpc
- route 53
- application load balancer

#### 배포도구
- bash script를 이용한 jar 파일를 배포합니다.
- 나중에는 배포 app을 이용할 예정입니다.
